Transient Review Tool

Candidate Serum-Style Phase Warp Visualizer

This file does not use the current synth code. It is a standalone visualizer for the candidate transfer equations for Bend +/-, PWM, Asym +/-, and Mirror. The goal is to let you drag the knob and judge whether the transfer behavior matches what you expect before we put any of it back into the oscillator.

The mode descriptions below are based on the public wording in the Serum 2 User Guide. Xfer documents the behavior semantics, but not the exact internal equations. This visualizer therefore shows one coherent candidate equation set that matches those documented semantics.

Phase Transfer

Input phase on the x-axis, warped lookup phase on the y-axis. The diagonal is identity.

Saw Cycle

The blue line is a single raw saw cycle. The pink line is that same saw read at the warped phase.

More Waveforms

The same warp is shown against four source waves at once: sine, triangle, saw, and square.
