Transient Prototype

ADSR editor test bed

This is a standalone ADSR editor prototype, not wired into the synth. The graph is the main control surface, each stage handle is draggable, and the exact values stay editable below it. The time response slider on the right exists so we can tune the short-time feel before we move this into the app.
