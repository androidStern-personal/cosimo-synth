Variant A
Compact Toolbar + Dropdown List
The standard VST pattern. A narrow horizontal bar shows the active preset name with prev/next arrows. Clicking the name opens a dropdown with search, factory/user filter pills, and a scrollable preset list. Right-click any preset for context actions. This is what Surge, Vital, Diva, and FabFilter all use.
